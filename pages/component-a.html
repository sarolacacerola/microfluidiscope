Base and Sorting modules

Each sorting module acts as a 3D microfluidic chip that can be stacked on top of each other: you decide the modularity, and how many sorting processes you want to perform with your sample.

Bill of Materials

  • Thermoplastic polyurethane (TPU) Filament £11.99 · Amazon

How to build it

  1. 3D modelling & printing 3D modelling and printing of each sorting module. To 3D print, we used materials and methods presented in Nelson, Ramkumar & Gale, 2019.  STL files on GitHub  ·  Reference paper
  2. Assembling Carefully paste each sticker on the corresponding sorting module to correctly label it.  Sticker files on GitHub
  3. Stacking Stack the modules on top of each other. Make sure to stack them in the correct order: the top module has one spiral, then stack it on top of the module with two spirals, then the one with four spirals.
